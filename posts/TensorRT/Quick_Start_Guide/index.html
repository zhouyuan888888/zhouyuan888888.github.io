TensorRT Installation

There are many different ways to install TensorRT, such as:

  • using a container,
  • using a Debian file,
  • using a Tar file, and
  • using a standalone pip wheel file.
  • ....
    •

    However, in this blog, I am mainly focusing elaborating the steps about installing TensorRT through a Tar file. Notice that currently, the TensorRT installed based on a standalone pip wheel file can only support x86_64 CPU

    .

    # install rbenv, will also install ruby-build
$ brew install rbenv

# upgrade rbenv and ruby-build
$ brew upgrade rbenv ruby-build

# add `eval "$(rbenv init -)"` to your profile
$ echo 'eval "$(rbenv init -)"' >> ~/.zshrc

    bundler

    # install Ruby gems
$ gem install bundler

# manage gem dependencies
$ bundler install

# check the location where gems are being installed
$ gem env

    Intellij Idea / RubyMine

    Open a project from existing source and setup project SDK.

    • File → New → Project from existing sources…
    • Project Structure → Project → Project SDK: rbenv:3.0.2
    • Module → Ruby SDK and Gems: rbenv:3.0.2
